How To Create an Application in API Business Hub

Home Tutorials How To Create an Application in API Business Hub
  • Last Updated: 09-05-2024
  • Download this article as PDF

Before you begin your API testing, you must first create an application on the API Business Hub portal to obtain the API Key. The following are steps to create your API application.


1. Ensure you can view the APIs which you have requested access for. In this example, you have access to the Warehouse APIs.



2. Click My Workspace > Applications > Create New Application.



3. On the Create New Application window, fill in the details under Application info.



4. Click Add Products. Under Available Products, tick the API to add it to your application. Search for the API on the search bar if itâ€™s not listed under Available Products. Then click Create.



5. Your API application page, under Application Details, you can find your Application Key which you will use for your API testing. To copy the API Key, click the copy icon.



6. Your API application is created with an Application Key. Keep this key securely and do not share it to anyone. This will be required to consume the API.
